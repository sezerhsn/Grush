\# GOLDENRUSH-GRUSH



GOLDENRUSH-GRUSH şu aşamada iki gerçek omurgaya dayanır:



\- `contracts/`: `GRUSHToken`, `ReserveRegistry`, `RedemptionGateway`

\- `por/ + tools/ + transparency/`: bar list → Merkle root → attestation → publish/verify hattı



Not: `apps/`, `services/`, `explorer/` tarafı bu aşamada iskelet düzeyindedir. Bu repo bugün önce kontrat ve PoR deterministikliğini sağlamlaştırmak için kullanılıyor.
